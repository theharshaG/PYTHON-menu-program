# Python Menu Based Program
This is a beginner-level Python menu-driven program.
It performs different operations based on user choice.

## Features
1. Multiplication table
2. Even or Odd number check
3. Reverse number printing (10 to 1)
4. Multiples of 5 up to 100
5. Exit option

## How to Run
1. Install Python
2. Download the project
3. Open terminal
4. Run:
python menu-program.py

## Sample Output
Enter choice (1/2/3/4/5): 2  
Enter number: 6  
Even number

- if / elif / else
- for loops
